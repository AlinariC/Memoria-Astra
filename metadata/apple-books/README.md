# Apple Books Catalog

This catalog feeds `scripts/build_apple_itmsp.py` for the active Cinderwake packages.

- Provider short name: `AlinariCampbell`, matching the verified Transporter provider used in the archived Memoria Astra delivery lane.
- Vendor IDs use underscore-safe permanent IDs because no-ISBN Apple package identifiers previously failed when hyphenated.
- BISAC subjects are set for epic fantasy, fantasy action/adventure, and dragons/mythical creatures.
